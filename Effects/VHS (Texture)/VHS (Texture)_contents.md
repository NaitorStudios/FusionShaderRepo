## Shader files
- **VHS (Texture).xml** — Info & Integration
- **VHS (Texture).fx** — DirectX9 source code
- **VHS (Texture).hlsl** — DirectX11 source code
- **VHS (Texture).fxc** — DirectX11 compiled
- **VHS (Texture).premultiplied.fxc** — DirectX11 compiled (Premultiplied*)
- **VHS (Texture).fxao** — Android source code
- **VHS (Texture).fxmo** — iOS/Mac source code
*Sometimes a shader needs Premultiplied version to fix Alpha Channel handling in Premultiplied graphics.*
