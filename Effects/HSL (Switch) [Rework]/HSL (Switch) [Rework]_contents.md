## Shader files
- **HSL (Switch) [Rework].xml** — Info & Integration
- **HSL (Switch) [Rework].fx** — DirectX9 source code
- **HSL (Switch) [Rework].hlsl** — DirectX11 source code
- **HSL (Switch) [Rework].fxc** — DirectX11 compiled
- **HSL (Switch) [Rework].premultiplied.fxc** — DirectX11 compiled (Premultiplied*)
- **HSL (Switch) [Rework].fxao** — Android source code
- **HSL (Switch) [Rework].fxmo** — iOS/Mac source code
*Sometimes a shader needs Premultiplied version to fix Alpha Channel handling in Premultiplied graphics.*
