## Shader files
- **3Denhanced.xml** — Info & Integration
- **3Denhanced.fx** — DirectX9 source code
- **3Denhanced.hlsl** — DirectX11 source code
- **3Denhanced.fxc** — DirectX11 compiled
- **3Denhanced.premultiplied.fxc** — DirectX11 compiled (Premultiplied*)
*Sometimes a shader needs Premultiplied version to fix Alpha Channel handling in Premultiplied graphics.*
