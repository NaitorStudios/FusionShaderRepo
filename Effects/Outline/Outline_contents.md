## Shader files
- **Outline.xml** — Info & Integration
- **Outline.fx** — DirectX9 source code
- **Outline.hlsl** — DirectX11 source code
- **Outline.fxc** — DirectX11 compiled
- **Outline.premultiplied.fxc** — DirectX11 compiled (Premultiplied*)
*Sometimes a shader needs Premultiplied version to fix Alpha Channel handling in Premultiplied graphics.*
