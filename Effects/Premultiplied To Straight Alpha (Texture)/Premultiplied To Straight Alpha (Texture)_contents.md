## Shader files
- **Premultiplied To Straight Alpha (Texture).fx** — DirectX9 source code
- **Premultiplied To Straight Alpha (Texture).hlsl** — DirectX11 source code
- **Premultiplied To Straight Alpha (Texture).fxc** — DirectX11 compiled
- **Premultiplied To Straight Alpha (Texture).premultiplied.fxc** — DirectX11 compiled (Premultiplied*)
- **Premultiplied To Straight Alpha (Texture).fxao** — Android source code
- **Premultiplied To Straight Alpha (Texture).fxmo** — iOS/Mac source code
*Sometimes a shader needs Premultiplied version to fix Alpha Channel handling in Premultiplied graphics.*
