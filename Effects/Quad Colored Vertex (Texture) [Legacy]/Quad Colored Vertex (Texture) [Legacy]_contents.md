## Shader files
- **Quad Colored Vertex (Texture) [Legacy].fx** — DirectX9 source code
- **Quad Colored Vertex (Texture) [Legacy].hlsl** — DirectX11 source code
- **Quad Colored Vertex (Texture) [Legacy].fxc** — DirectX11 compiled
- **Quad Colored Vertex (Texture) [Legacy].premultiplied.fxc** — DirectX11 compiled (Premultiplied*)
- **Quad Colored Vertex (Texture) [Legacy].fxao** — Android source code
- **Quad Colored Vertex (Texture) [Legacy].fxmo** — iOS/Mac source code
*Sometimes a shader needs Premultiplied version to fix Alpha Channel handling in Premultiplied graphics.*
