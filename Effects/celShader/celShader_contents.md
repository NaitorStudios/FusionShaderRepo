## Shader files
- **celShader.xml** — Info & Integration
- **celShader.fx** — DirectX9 source code
- **celShader.hlsl** — DirectX11 source code
- **celShader.fxc** — DirectX11 compiled
- **celShader.premultiplied.fxc** — DirectX11 compiled (Premultiplied*)
*Sometimes a shader needs Premultiplied version to fix Alpha Channel handling in Premultiplied graphics.*
