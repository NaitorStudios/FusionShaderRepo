## Shader files
- **cd_texture.xml** — Info & Integration
- **cd_texture.fx** — DirectX9 source code
- **cd_texture.hlsl** — DirectX11 source code
- **cd_texture.fxc** — DirectX11 compiled
- **cd_texture.premultiplied.fxc** — DirectX11 compiled (Premultiplied*)
*Sometimes a shader needs Premultiplied version to fix Alpha Channel handling in Premultiplied graphics.*
