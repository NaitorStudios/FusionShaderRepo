## Shader files
- **CRT2.xml** — Info & Integration
- **CRT2.fx** — DirectX9 source code
- **CRT2.hlsl** — DirectX11 source code
- **CRT2.fxc** — DirectX11 compiled
- **CRT2.premultiplied.fxc** — DirectX11 compiled (Premultiplied*)
*Sometimes a shader needs Premultiplied version to fix Alpha Channel handling in Premultiplied graphics.*
